# Schemas

Machine-readable schemas for ModelSpec serializations.

The ModelSpec repository is the source of truth for schema files. The website should
publish stable copies for consumers.

## Planned Artifacts

| File | Stable URL | Purpose |
|---|---|---|
| `modelspec-ast.schema.json` | `https://modelspec.org/schema/modelspec-ast.schema.json` | Latest compatible ModelSpec AST JSON Schema. |
| `modelspec-ast-1.0-draft.schema.json` | `https://modelspec.org/schema/modelspec-ast-1.0-draft.schema.json` | Versioned JSON Schema for the `1.0-draft` AST serialization. |
